This will construct a label using an array representation of a path (path_hops). If any label along the path needs to be re-encoded, it will be.

get_encoding_form

Get the encoding form used for the first director of the RoutingLabel. It also returns index of found form in scheme. Recall an encoding scheme is one or more encoding forms. If the label is not recognized as using the given scheme then it'll return Err(Error::CannotFindForm).

is_one_hop

Tests if a label contains only one hop.

splice
